# animals-rest-go
Simple REST-API with use Docker, PostgreSQL and Golang (GIN & Gorm)

## Getting Started

### Configuration

Yout should create ```postgres.evn``` file with next envs:

```
    DB_HOST="xxx"
    DB_PORT=0000

    POSTGRES_PASSWORD="xxx"
    POSTGRES_USER="xxx"
    POSTGRES_DB="xxx"
```

### Run docker compose

```
docker-compose up -d
```
