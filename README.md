# animals-rest-go
Simple REST-API with use ***Docker, PostgreSQL and Golang (GIN & Gorm)*** and ***Test-containers*** for testing

## Getting Started

### Configuration

You should create env files by example from ***docker/env/.env.example*** file

### Run make command

```
make run-dev-app
```
OR
```
make run-prod-app
```

### Run tests

```
make run-tests
```
